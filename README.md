# 💻 Computer Basics Quiz App
Welcome to the **Computer Basics Quiz App**, an interactive and engaging web-based quiz application designed to test and improve users’ knowledge of fundamental computer concepts.
This project is built using **HTML, CSS, and JavaScript**, and provides a smooth quiz experience with a modern user interface, countdown timer, automatic answer checking, instant feedback, and score tracking.

Whether you are a beginner learning computer basics or someone preparing for exams/interviews, this quiz app helps strengthen essential computer knowledge in a fun and practical way.
--- 
# ✨ Project Overview

The Computer Basics Quiz App is a dynamic multiple-choice question platform where users answer computer-related questions within a limited time.
Each question is displayed one at a time with:
- A countdown timer of 15 seconds
- Four multiple-choice options
- Instant correct/wrong answer highlighting
- Automatic submission if time runs out

At the end of the quiz, users receive:
- Final score summary
- Performance-based motivational message
- Emoji reward feedback

# 🚀 Features

### 🎯 Interactive Quiz System
- 10 computer basics questions included

# 🧠 How It Works

### Step 1:
Quiz starts automatically when page loads.

### Step 2:
Question appears with four answer options.

### Step 3:
User selects one answer before timer ends.

### Step 4:
System checks answer instantly.

### Step 5:
Score updates automatically.

### Step 6:
After last question, final result is shown.
